# organ

This is a React Native app built with Expo. To run it locally, run `npx expo start`.

Currently the app displays a static iCal calendar. Planned features:
- store list of iCal feed links and sync events with local store
- login to Matrix server to store account data
- social events features on Matrix - RSVPs, event updates & announcements, subscribe to event hosts, discover new hosts and events
- interoperate with email, SMS
